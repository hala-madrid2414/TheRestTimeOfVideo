# B站视频剩余时长计算器

这是一个Chrome浏览器扩展，帮助B站用户计算合集视频的剩余总时长。对于那些包含多集内容的长课程视频，用户可以通过此扩展快速了解还需要多少时间才能看完整个系列。

## 安装方法

### 方法一：从Chrome网上应用店安装（尚未上架）

1. 访问Chrome网上应用店
2. 搜索"B站视频剩余时长计算器"
3. 点击"添加到Chrome"按钮

### 方法二：开发者模式安装（当前推荐）

1. 下载并解压本扩展文件到本地文件夹
2. 打开Chrome浏览器，在地址栏输入：`chrome://extensions/`
3. 在右上角开启"开发者模式"（Developer mode）
4. 点击左上角的"加载已解压的扩展程序"（Load unpacked）按钮
5. 选择包含本扩展所有文件的文件夹（即包含manifest.json的文件夹）
6. 扩展将被添加到Chrome浏览器中

## 使用方法

1. 安装扩展后，在Chrome浏览器的工具栏中会出现扩展图标
2. 访问任意B站合集视频页面（例如：https://www.bilibili.com/video/BV1xx411c7mD）
3. 当视频开始播放后，点击工具栏中的扩展图标
4. 弹出窗口将显示以下信息：
   - 当前视频标题
   - 当前播放进度
   - 剩余视频数量
   - 剩余总时长
   - 预计完成时间
5. 如需刷新数据，点击窗口底部的"刷新数据"按钮

## 注意事项

- 本扩展仅在B站视频页面有效
- 计算结果基于当前页面中可见的视频列表
- 预计完成时间基于正常播放速度（1.0x）计算
- 如遇到数据不准确，请尝试刷新页面后再使用

## 常见问题

**Q: 为什么扩展显示"无法计算剩余时长"？**  
A: 可能原因：
1. 当前页面不是B站视频页面
2. 页面结构发生变化，扩展无法识别
3. 视频不是合集形式

**Q: 预计完成时间是如何计算的？**  
A: 基于当前系统时间加上剩余视频的总时长

**Q: 扩展支持哪些类型的B站视频？**  
A: 目前主要支持常规的视频合集，部分课程和番剧可能支持不完善

## 隐私说明

本扩展不会收集任何个人数据，所有计算均在本地完成。扩展仅在B站视频页面上运行，且只读取视频时长相关信息。
